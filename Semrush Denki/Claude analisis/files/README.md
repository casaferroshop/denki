## 💡 CONSEJOS PRÁCTICOS

### Para Crear Landing Pages:

✅ **DO:**
- Usa el template completo proporcionado
- Pide revisión médica por nutrióloga
- Incluye 3+ CTAs por página
- Agrega testimonios reales
- Cita fuentes médicas autorizadas

❌ **DON'T:**
- Copiar texto de otros sitios (plagio)
- Hacer promesas absolutas ("cura")
- Publicar sin revisar ortografía
- Olvidar Schema markup
- Ignorar mobile optimization

### Para Crear Blog Posts:

✅ **DO:**
- Usa templates proporcionados
- Sé práctico y accionable
- Incluye listas, tablas, infografías
- Link a landing pages relacionadas
- Actualiza fechas anualmente

❌ **DON'T:**
- Escribir muros de texto
- Usar lenguaje muy técnico
- Olvidar CTAs
- Ignorar internal linking
- Publicar posts <1,000 palabras

### Para SEO Local:

✅ **DO:**
- Completa GMB 100%
- Solicita reseñas constantemente
- Publica en GMB semanalmente
- Responde todos los reviews
- Usa fotos reales del consultorio

❌ **DON'T:**
- Comprar reseñas falsas
- Ignorar reviews negativos
- Usar fotos de stock
- Inconsistencia en NAP (nombre, dirección, teléfono)
- Abandonar GMB después de configuración inicial
